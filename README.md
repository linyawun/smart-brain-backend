# smart-brain-backend
> 此為 [smart-brain](https://github.com/linyawun/smart-brain)的後端，提供前端 API ，並連接 PostgreSQL 資料庫。

> 此後端與資料庫皆運行在 [Zeabur](https://zeabur.com/zh-TW)。

## 功能 
- [x] 註冊 
- [x] 登入
- [x] 取得 user 資料
- [x] 呼叫 clarifai face detection 並回傳辨識資料

## 專案技術
- Node.js v18.12.0
- express v4.18.2
- bcryptjs v2.4.3
- clarifai v2.9.1
- knex v2.5.1
- node-fetch v3.3.2
- pg v8.11.2
